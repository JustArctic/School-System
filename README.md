This project is a web-based enterprise school management system developed using:
Python

The system allows students to:
securely log in
view their personalised timetable
search subjects to find classroom locations
access school events
browse canteen items and prices

The application includes:
🔐 Login authentication system
🗂️ Database-driven timetable
🏫 Classroom and block directory
🎉 Events management page
🍔 Interactive canteen menu
🔎 Subject/classroom search functionality
🎨 Blue and yellow school-themed user interface

The system demonstrates enterprise computing concepts including:
database management
dynamic web development
user authentication
data relationships
frontend and backend integration
user interface design
